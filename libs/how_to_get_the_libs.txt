You don't need to build GIMP to get .lib files. Just grab the .def files from the source package, put them in the same folder as the compiled DLLs, then open the Visual Studio Command Prompt and run this command:

lib /def:libwhatever.def /out:libwhatever.lib /machine:x86

To build gimp-vtf you will need:

intl
libgimp-2.0-0
libgimpbase-2.0-0
libgimpui-2.0-0
libgimpwidgets-2.0-0
libglib-2.0-0
libgobject-2.0-0
libgtk-win32-2.0-0
